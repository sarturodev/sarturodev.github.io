See installation instructions for Chrome, Brave and Opera

Chrome

  • 1- Download the file bayhelper.zip , and unzip the folder bayhelper.

  • 2- At the top right of your browser, click on More > More tools > Extensions.

    or type in the address bar chrome://extensions/

  • 3- Enable the Developer Mode, by clicking the button at the top right.

  • 4- Drag the folder bayhelper into the browser, previously unzipped in step 1.

Brave

  • 1- Download the file bayhelper.zip , and unzip the folder bayhelper.

  • 2- At the top right of your browser, click on More > More tools > Extensions.

    or type in the address bar chrome://extensions/

  • 3- Enable the Developer Mode, by clicking the button at the top right.

  • 4- Drag the folder bayhelper into the browser, previously unzipped in step 1.

Opera

  • 1- Download the file bayhelper.zip , and unzip the folder bayhelper.

  • 2- Click on the Opera Main Menu > Extensions > Extensions.

    or type in the address bar opera://extensions/

  • 3- Enable the Developer Mode, by clicking the button at the top right.

  • 4- Drag the folder bayhelper into the browser, previously unzipped in step 1.
