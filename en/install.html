See installation instructions for Opera, Chrome and Brave

Opera

  • 1- Download the file bayhelper.nex.

  • 2- In the message at the top of your browser, click the Go to Extensions button.

    or type in the address bar opera://extensions/

  • 3- Enable the Developer Mode, by clicking the button at the top right.

  • 4- Click the Install button, then click Yes, Install button.

  • If the message of Step 2 does not appear, go to the Opera Extensions page and drag the file bayhelper.nex into the browser and continue with Step 4.

Chrome

  • 1- Download the file bayhelper.zip, and unzip the folder bayhelper.

  • 2- At the top right of your browser, click More > More tools > Extensions.

    or type in the address bar chrome://extensions/

  • 3- Enable the Developer Mode, by clicking the button at the top right.

  • 4- Drag the folder bayhelper into the browser, previously unzipped in step 1.

Brave

  • 1- Download the file bayhelper.zip, and unzip the folder bayhelper.

  • 2- At the top right of your browser, click More > More tools > Extensions.

    or type in the address bar chrome://extensions/

  • 3- Enable the Developer Mode, by clicking the button at the top right.

  • 4- Drag the folder bayhelper into the browser, previously unzipped in step 1.
